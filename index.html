We have different tracks with sessions about Cloud, DevOps, AI, Machine Learning, Blockchain, Open Hardware, Science, Web, Mobile and more.

Topics

Artificial Intelligence

From building an AI server infrastructure to creating skills and actions for voice assistants or create your own Open Source chatbots with SUSI.AI. Join sessions covering Bot Framework, Big Data, Dataprep and more.

Cloud, Container, DevOps

Get better insights in how to deploy apps on the cloud more efficiently and save your company money. Sessions in the Cloud, Containers, DevOps track cover Kubernetes, Serverless Architecture, Microservices, OpenShift, and Continuous Integration.

Database

Meet database engineers and learn how to save your company money with Open Source database solutions. Specific topics include Scaling TB of data, Real-time data masking, Replication Features, Performance Schema, BigQuery, and Immutable Key-Value Stores.

Mobile Technologies

Mobile development sessions covers technologies for mobile platforms, Android, iOS, cross platform tools and discuss current trends and future of mobile technologies.

Web Technologies

Web development sessions covers technologies for web platforms, Javascript, HTML, frameworks, development tools as well as upcoming trends and technologies.

Hardware, Design, Firmware

With the decrease of prices of hardware development tools and production open source hardware projects are gaining popularity. Projects range from laser cutters to pocket science labs. Apart from technical topics speakers will share their experience from making prototype to going to large scale production and working with mass manufacturers.

Kernel & Platform

Topics from the Linux Kernel to BSD and desktop systems such as Meilix or applications like VLC. Learn about network servers, hacking with x86 Windows tablets, asynchronous integration of GPU computing with HPX many task processing, Unikernelized Linux, Open Build Service in Debian, and C from the trenches.

Science & Education

Experimenting with Pocket Science Lab

The goal of Pocket Science Lab is to create an Open Source hardware device (open on all layers) that can be used for experiments by teachers, students and citizen scientists. This tiny pocket lab provides an array of sensors for doing science and engineering experiments. It comes with functions of numerous measurement devices including an oscilloscope, a waveform generator, a frequency counter, a programmable voltage, current source and as a data logger.

About OpenTechSummit

OpenTechSummits are premier Free and Open Source technology events for developers, educators, designers, businesses, NGO and governmental organizations. The EU Summit is organized by the TU Berlin and OpnTec. Our mission is to improve people's lives by sharing open technologies, knowledge and resources and building a sustainable FOSS ecosystem. Topics at events range from open source software, to hardware, and design covering areas such as artificial intelligence, cloud and containers, Blockchain, programming languages like Python, operating systems, open science, open education and more. The first open tech event took place in 2009 in Vietnam. Other locations of regular summits are across Asia and Europe in Austria, Cambodia, China, Germany, India, Japan, Malaysia, and Thailand. At many events we run coding programs, hackathons and development contests like Codeheat and support new contributors to join. To organize events we develop an open source event management platform.
